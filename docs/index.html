Grammar-inspired plotting in C

Publication-ready graphics
from one small, fast binary.

cinderplot turns a CSV into a PDF or SVG with a ggplot2-inspired grammar — panels, hue palettes, scales, legends — rendered by a single C binary with Cairo. No R, no Python, no runtime at plot time.

See the gallery GitHub 
# install from the zhou-lab conda channel
conda install -c zhou-lab cinderplot

# plot straight from a CSV
cinderplot 'data.csv + aes(hp, mpg, colour=factor(cyl)) + geom_point()' -o plot.pdf

Small & fast

One C11 binary, Cairo for output. Milliseconds per figure, minimal dependencies.

Familiar grammar

aes, geoms, scales, and legends inspired by ggplot2 — reproduces theme_gray closely.

PDF & SVG

Vector output chosen by file extension. Publication-ready, no rasterisation.
