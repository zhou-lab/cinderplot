cinderplot

Grammar-inspired plotting in C

Publication-ready graphics from one small, fast binary.

cinderplot turns a CSV into a PDF or SVG with a ggplot2-inspired grammar — panels, hue palettes, scales, legends — rendered by a single C binary with Cairo. No R, no Python at plot time.

Install & plot

# from the zhou-lab conda channel
conda install -c zhou-lab cinderplot

# plot straight from a CSV
cinderplot 'data.csv
  + aes(hp, mpg, colour=factor(cyl))
  + geom_point()' -o plot.pdf

Why cinderplot

  • Small & fast. One C11 binary, Cairo output. Milliseconds per figure, no runtime.
  • Familiar grammar. aes, geoms, scales, legends and themes inspired by ggplot2 — clean theme_minimal default, theme_bw/classic/void and more on tap.
  • Vector out. PDF or SVG chosen by file extension. Publication-ready, no rasterisation.
